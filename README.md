# Portfolio CMS

**A self‑hosted portfolio content management system for freelancers, developers, and small agencies.**

---

## Features

- Public portfolio pages (Home, About, Projects, Blog, Contact)
- Secure admin dashboard with project & blog CRUD
- JWT authentication via httpOnly cookies with silent refresh
- Markdown blog with live preview editor
- Image upload and responsive Tailwind CSS design

---

## Tech Stack

**Frontend:** React (Vite) · React Router · Axios · Tailwind CSS  
**Backend:** Node.js · Express · PostgreSQL · JWT · Multer  
**Deployment:** Vercel (Frontend) · Render (Backend + DB)
